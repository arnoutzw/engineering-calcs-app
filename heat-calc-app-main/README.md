# VDI Heat Transfer Calculator

An engineering heat transfer calculator based on VDI Heat Atlas correlations for accurate thermal analysis and design calculations.

## Features

- **Heat Transfer Modes**:
  - Forced convection (internal and external flows)
  - Free (natural) convection
  - Conduction analysis
  - Radiation calculations
- **Advanced Correlations**:
  - Circular tube flows with Gnielinski correlation
  - Multiple flow regime support
- **Flow Analysis**:
  - Laminar flow calculations
  - Turbulent flow analysis
  - Automatic flow regime detection
- **Temperature Inputs**:
  - Bulk temperature definition
  - Wall temperature specification
  - Temperature-dependent property evaluation
- **Heat Exchanger Analysis**: Design and optimization of heat exchange systems
- **VDI Standards**: Calculations based on internationally recognized VDI Heat Atlas methods
- **Offline Support**: Full functionality without internet connection
- **Engineering Accuracy**: Precise correlations for professional thermal design

## Tech Stack

- **HTML5** - Semantic markup
- **JavaScript** - Thermal engineering calculations and correlation implementations
- **PWA** - Progressive Web App capabilities with offline support

## Getting Started

### Online Demo
Visit the live application: [https://arnoutzw.github.io/heat-calc-app/](https://arnoutzw.github.io/heat-calc-app/)

### Local Installation
Simply open `index.html` in a modern web browser. No build tools or dependencies required.

### Usage
1. Open the application in your browser
2. Select the heat transfer mode (convection, conduction, radiation)
3. Choose the relevant correlation or flow type
4. Enter your system parameters:
   - Fluid properties
   - Dimensions
   - Temperatures
5. View calculated results with proper engineering units
6. Use results for thermal design and system optimization

## Available Correlations

- **Forced Convection**: Gnielinski, Sieder-Tate, Churchill, and others
- **Free Convection**: Correlations for vertical and horizontal surfaces
- **Conduction**: Steady-state and transient calculations
- **Radiation**: View factor and emissivity considerations

## Browser Support

Requires a modern browser with JavaScript support (Chrome, Firefox, Edge, Safari)
